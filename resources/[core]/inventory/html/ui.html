Help:
 Para mover uma pilha completa, verifique se o AMOUNT FIELD está vazia.Você pode digitar o específico NUMBER AMOUNT Por quanto você gostaria de se mudar.Para USAR um item, arraste -o para a opção do meio intitulada "USAR" ou você pode DOUBLE CLICK. o AMOUNT FIELD Não será redefinido quando você fechar seu inventário, portanto, tenha o hábito de limpar o número!
Middle Options:
  • Use: Arraste o item para este botão para usar esse item
  • Give: Dê um item a outros cidadãos ao seu redor
  • Combine: Combine certos itens
  • Switch: Switch Item atual
  • Attachments: Quando você tem itens, você pode anexar às suas armas
  • Close: Outra maneira de fechar o inventário
Keybinds (default):
  • Left Click: Arrasto normal
  • Right Click: Soltar item ou mover armazenamento de item único
  • Double Click: Usar Item
  • Number 0-9: Inserirá o número digitado
  • BACKSPACE Key: Irá definir AMOUNT para blank/move Item inteiro pilha
